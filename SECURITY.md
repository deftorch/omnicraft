## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please follow these guidelines:

1. **Do NOT report security vulnerabilities through public GitHub issues.**
2. Send an email to **security@omnicraft.dev**.
3. Include as much information as possible:
   - Type of vulnerability
   - Steps to reproduce
   - Potential impact
   - Code samples if applicable

### Response Timeline

- You should receive an acknowledgment within **48 hours**.
- We will provide a timeline for fixing the vulnerability within **1 week**.
- Once fixed, we will coordinate a public disclosure and release.

### Bounty Program

We currently do not offer a paid bug bounty program, but we will publicly acknowledge your contribution in our release notes and Hall of Fame (with your permission).
